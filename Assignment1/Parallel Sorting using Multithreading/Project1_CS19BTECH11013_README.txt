
  README FILE FOR C PROGRAM FOR PARALLEL SORTING USING MULTI-THREADING:
  
   1)Firstly,we need to open terminal and start writing the program by opening the text editor which will open when you type "gedit filename.c".(Here filename is name of file)
   
   2)After writing the program completely, save the program by pressing "ctrl+s".
   
   3)Now open the terminal and type "gcc filename.c -o filename -lpthread" and press "enter" to compile the program.
   
   4)Then type "./filename" and press "enter" to run and execute the compiled program.
